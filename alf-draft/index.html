💬 FAQ — 단순·반복 질문에 효과적

형식 규정 (채널톡 공식)

  • 질문: 최대 100자
  • 추가 질문: 최대 100자, 최대 10개
  • 답변: 최대 500자
  • 등록 개수 제한 없음

ALF 동작 방식

"등록된 FAQ의 질문과 관련도가 높은 문의가 인입될 경우, FAQ의 답변을 참고하여 대답합니다."

→ 공개(published) 상태의 FAQ만 참조됩니다.

효과적인 작성 팁

"질문에는 정확한 키워드를 입력해 주시고, 추가 질문은 질문과 키워드를 포함해 다양하게 설정해 주세요."
"답변을 구체적으로 입력해 주셔야, 추후 ALF가 더 잘 활용할 수 있습니다."
  • 고객이 실제로 검색할 표현으로 질문 작성
  • 추가 질문엔 줄임말·구어체·오타까지 (예: "환불 어케해요?")
  • 답변은 결론부터, 조건 분기는 명확히

📄 아티클 — 복잡·다단계 시나리오에 효과적

ALF가 활용하는 방식

"지식이 잘 정리되어 있을수록 RAG 품질이 향상되며, 구체적이고 구조화된 지식을 추가할수록 ALF는 더욱 정확하고 신뢰도 높은 답변을 제공합니다."

효과적인 구조

  • 제목: 고객이 검색할 표현 — # 구독 중인 플랜을 환불받는 방법
  • 첫 문단 = 결론 — ALF는 앞부분을 우선 참조해서 답변 생성
  • 섹션 분리: ## 헤딩으로 케이스/단계 구분
  • 조건 분기 명시: "경우에 따라" 대신 케이스별로
    • 결제 후 7일 이내 → 전액 환불
    • 7일 ~ 30일 → 50% 환불
  • 시각 요소: 비교는 표(|), 강조는 콜아웃(>), 절차는 번호 목록
  • 2,000자 이내 권장: 초과 시 별도 아티클로 분리
  • 아티클 안에 FAQ 첨부 가능: 큰 가이드 + 자주 묻는 짧은 질문 조합

피해야 할 표현

  • ❌ "보통", "일반적으로", "경우에 따라"
  • ❌ "담당자에게 문의하세요" 단독 사용 → 조건/방법 같이 명시
