# 상품/패키지 생성 및 강의 구성

> 원본: [노션 PRD 링크](https://www.notion.so/futureschole/PRD-333c7a52db4f80bbb456c778f04e5770) | 최종 갱신: 2026-06-15

<!-- TL;DR -->
> **상품 구조**: LNB 개편 + 상품/패키지 생성~구성 한 흐름. LK1.0의 퍼널 단절/정보 혼재/패키지 진입점 부재 해결.
> 상품 타입: 강의/라이브/디지털상품/코칭. 상태: 설정 중(임시 저장)→판매 중→판매 중단. 노출 설정: 전체 공개/일부 공개. 타입별 아이템 제약.
> 관리 목록 탭: 전체/카테고리는 모든 상태 노출 + 상태 필터; 비공개·휴지통 탭 유지. [프로그램] 페이지 생성 버튼 유지.
> 판매 시작 필수항목: 제목 + 가격 + 수강기간. 패키지 구성 상품 수 제약 해제(2개 이상 불필요). 복사/삭제/카테고리 관리. 미리보기 지원.
> 권한별 접근: 소유자/운영자는 LNB 전체 접근; 강사는 LNB-less로 본인 상품만 접근. 관련: offer, options, membership_subscription

## 개요

크리에이터가 사이트 관리 페이지 내에서 상품과 패키지를 **흐름이 끊기지 않고, 혼선 없이** 생성하고 강의 구성을 완료할 수 있도록 개선한다.

### 배경/문제

1. **상품 생성/설정/관리 퍼널의 단절** — 상품 생성, 세부 설정, 생성된 상품 관리가 서로 다른 퍼널로 분리되어 있어, 크리에이터가 하나의 상품을 완성하기까지 맥락이 끊기는 UX가 반복됨.
2. **상품 설정 페이지 내 정보 혼재** — 상품 기본 정보(제목, 설명, 썸네일 등), 강의 구성, 오퍼 설정(모집 정원, 모집 기간, 수강 기간, 가격 등)이 하나의 설정 화면에 혼재되어 있어 크리에이터가 어느 단계에서 무엇을 설정해야 하는지 혼란을 야기함.
3. **패키지 생성 진입점 부재** — 패키지 생성 버튼이 '프로그램' 페이지에서만 노출되며, 해당 페이지가 LNB 메뉴에서 숨겨진 경우 크리에이터가 패키지 생성 자체를 찾지 못하는 문제가 발생함.

### 개선 목표

1. LNB 구조 변경으로 상품 유형별 진입점 명확화
2. 상품/패키지 생성~구성 완료까지 한 흐름으로 연결
3. 상품/패키지 관리 목록 UX/UI 개선

2. 2depth 메뉴 구성 변경

   2. 기존 4개 유형 커리큘럼을 상품으로 분리
   3. 패키지 메뉴 추가

### [3] 상품/패키지 상태

**판매 상태**

| 상태 | 설명 |
|------|------|
| 설정 중(임시 저장) | 생성 직후 기본 상태. 한 번도 판매 중 상태로 전환된 적이 없음 |
| 판매 중 | 구매 가능 상태. 노출 설정에 따라 사이트 노출 여부 결정 |
| 판매 중단 | 구매 불가 상태. 기존 구매자 수강 권한 유지. 노출 설정에 따라 사이트 노출 여부 결정 |

**노출 설정** (판매 중/판매 중단 상태에 적용)

| 노출 설정 | 설명 |
|----------|------|
| 전체 공개 | 사이트에 노출됨 (LK1.0 기준 공개) |
| 일부 공개 | 사이트에 미노출 (LK1.0 기준 비공개) |

### [4] 상품 생성 Flow 변경

| as-is (LK1.0) | to-be (LK2.0) |
|----------------|---------------|
| 1. 상품 생성 페이지 진입 | 1. 상품 생성 페이지 진입 |
| 2. 커리큘럼 유형 선택 → 세부 설정 항목 중 일부 설정 | 2. **상품 제목만 입력하면 상품이 생성**됨 |
| 3. 일부 세부 항목 설정 완료 후 **상품이 비공개로 생성** | 3. 생성된 상품은 **'설정 중(임시 저장)' 상태** 값을 가짐 |

### [5] 공개 전환 필수 항목

- **상품**: 상품 제목, 무료 또는 유료+정가, 수강기간
- **패키지**: 패키지 제목, 정가, 수강기간

### [6] 삭제 정책

- 다중 선택 삭제 지원
- 패키지 삭제 시 포함 상품은 유지
- 삭제된 상품/패키지는 휴지통으로 이동하여 7일간 보관

### [7] 복사 정책

- 상품/패키지 복사 시 이름 설정 가능
- 리뷰 복사 여부 선택 가능
- 복사본은 '설정 중' 상태로 생성

### [8] 카테고리

- 카테고리 생성/추가/제거/삭제 가능
- 1개 상품에 N개 카테고리 연결 가능
- 카테고리 삭제 시 해당 카테고리에 속한 상품/패키지는 삭제되지 않음

### [9] 상품 설정 페이지 진입 위치 변경

- LK1.0: 고객사 사이트(`/classes/*/settings`)에서 접근
- LK2.0: 사이트 관리 페이지(어드민) 내에서 직접 접근

### [10] 상품 관리 목록 탭별 필터 정책

| 탭 | 노출 상품 | 비고 |
|----|----------|------|
| 전체 | 설정 중·판매 중·판매 중단 모든 상품 | 별도 "상품 상태" 필터로 상태 필터링 가능 |
| 카테고리 | 설정 중·판매 중·판매 중단 모든 상품 | 별도 "상품 상태" 필터로 상태 필터링 가능 |
| 비공개 | 비공개 상태 상품만 | LK1.0 정책 유지 |
| 휴지통 | 삭제된 상품 | LK1.0 정책 유지 |

### [11] [프로그램] 페이지 기능 유지 정책

기존 사용 경험 전환을 고려하여 MVP 이후 기간을 두고 fade out 예정.

- [프로그램] 페이지 클래스/패키지 생성 버튼 유지 → 클릭 시 사이트 관리 페이지 해당 생성 flow로 연결
- [프로그램] 페이지 비공개·휴지통 탭 유지

### [12] 패키지 구성 상품 최대 수

- 패키지에 포함할 수 있는 상품의 최대 개수는 200개이다.
- 200개를 초과하여 상품 추가 시도 시, 최대 200개까지 상품을 추가할 수 있다는 안내가 노출된다.

## 하지 않을 것 (Won't)

1. 새로운 상품 아이템 유형(LK1.0 기준, 커리큘럼 유형) 및 유형별 세부 설정 값에 신규 설정 값 추가하지 않는다.
   1. 라이브미팅, VOD, 디지털 상품, 오프라인 모임 각 유형의 세부 설정 항목 및 입력 값의 종류는 기존과 동일하게 유지한다.
   2. 새로운 설정 항목의 추가, 기존 설정 항목의 제거는 이번 범위에서 진행하지 않는다.
   3. 기존 4가지 상품 아이템 유형(VOD, 라이브미팅, 디지털 상품, 오프라인 모임) 외 신규 유형도 추가하지 않는다.
2. 기존 클래스/패키지 설정 페이지 내 각 탭에서 제공하던 항목을 제거하지 않는다.
   1. 기존 클래스/패키지 설정 페이지([수강생] / [일반 설정] / [강의 설정] / [결제 설정] / [추가정보] / [고급설정])에서 크리에이터가 사용할 수 있었던 모든 기능 및 설정 항목은 이번 개선 범위에서 그대로 유지한다.
